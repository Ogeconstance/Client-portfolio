Non-Humanitarian Project

Alfred (AI-Powered
Assistant)

Lack of sense of direction, and inability to locate classes
and venues in school grounds.

Methodology

An LLM-powered virtual assistant that interprets building plans and provides directions and assistance to users using combinatorial optimization, network analysis and natural language processing. The virtual assistant is also given access to custom built knowledge pool by integrating LangChain and Vector Stores into the solution. The enable the virtual to also answer general questions about the institution is built for.


Tools:

OpenAI GPT‑3
LangChain
Pinecone Vector Store
Telebot
Prompt Engineering
